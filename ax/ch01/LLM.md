# LLM (Large Language Model)

## 1. LLM이란?
LLM은 **대규모 언어 모델**(Large Language Model)의 약자로, 방대한 양의 텍스트 데이터를 학습하여 인간의 언어를 이해하고 생성할 수 있는 인공지능입니다.

대표적인 LLM 아키텍처로는 **GPT**(Generative Pre-trained Transformer)가 있으며, 이는 "생성형 사전 학습 트랜스포머"를 의미합니다.

> 📖 [GPT 아키텍처 상세 설명 보기](./detail_docs/GPT.md)

## 2. 작동 원리
- **학습**(Training): 인터넷의 책, 기사, 웹사이트 등 수많은 문서를 읽고 단어와 문장의 패턴을 학습합니다.
- **예측**(추론, Inference): 입력된 질문이나 문장에 대해 가장 적절한 다음 단어를 예측하여 답변을 생성합니다.
- **맥락 이해**: 단순 키워드 매칭이 아닌, 문맥을 파악하여 자연스러운 대화가 가능합니다.

### 💡 단어의 의미를 어떻게 이해할까?
LLM은 단어들을 좌표평면에 점을 찍듯이 **다차원 공간에 배치**하여 의미를 이해합니다. 예를 들어, "사과"와 "apple"은 같은 의미이므로 서로 가까운 위치에, 과일 "사과"와 용서를 구하는 "사과(혹은 apology)"는 의미가 다르므로 먼 위치에 배치됩니다.

## 3. 주요 활용 분야
| 분야 | 활용 예시 |
|------|-----------|
| **문서 작성** | 이메일, 보고서, 기획안 초안 작성 |
| **정보 검색** | 복잡한 질문에 대한 답변 및 요약 제공 |
| **번역** | 다국어 번역 및 의역 |
| **코딩 지원** | 프로그램 코드 작성 및 오류 수정 |
| **교육** | 개념 설명, 학습 자료 생성 |

## 4. LLM 모델의 종류
LLM은 크게 **클라우드 API 모델**과 **오픈소스 모델** 두 가지로 나뉩니다.

## 6. 한계점
- **환각(Hallucination)**: 사실이 아닌 정보를 그럴듯하게 생성할 수 있습니다. ([세종대왕의 맥북프로 던짐 사건](https://v.daum.net/v/66Y7KNxvOQ))
- **학습 데이터 기준**: 특정 시점까지의 공개 데이터만 학습했기 때문에 최신 정보나 회사 내부 문서 등 비공개 정보는 알지 못합니다.
- **맥락(Context Window) 제한**: 한 번에 처리할 수 있는 텍스트 양에 제한이 있어, 매우 긴 대화나 문서는 초반부 내용을 잊어버릴 수 있습니다.
- **무상태성(Statelessness)**: 각 대화 세션이 독립적이어서 이전 대화 내용을 기억하지 못합니다. (일부 서비스는 대화 기록 저장 기능 제공)
- **편향(Bias)**: 학습 데이터에 포함된 편향(성별, 인종, 직업 등)이 답변에 반영될 수 있습니다.
- **보안 문제**: 개인정보 및 기업 데이터 유출 위험 등이 존재합니다.
- **윤리적 문제**: 가짜 뉴스 생성 등 악용 가능성, 저작권 문제 등이 존재합니다.

> **주의**: LLM의 답변은 참고용으로 활용하고, 중요한 결정이나 전문적인 정보는 반드시 검증이 필요합니다.
