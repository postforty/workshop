컴퓨터의 5가지 특성

  • 정확성 (Accuracy): 주어진 명령을 단 하나의 오차 없이 정확하게 수행합니다.
  • 신속성 (Speed): 방대한 양의 데이터를 사람이 상상할 수 없는 속도로 빠르게 처리합니다.
  • 대용량성 (Large Memory): 많은 양의 정보를 저장하고, 필요할 때 언제든 꺼내 쓸 수 있습니다.
  • 자동성 (Automation): 한 번 프로그래밍된 작업을 사람의 개입 없이 스스로 반복 수행합니다.
  • 융통성 (flexibility): 없습니다.

파이썬(Python)의 장점

쉬운 문법

영어와 비슷하고 코드가 간결하여 초보자도 빠르게 적응할 수 있음

높은 생산성

다양하고 강력한 라이브러리를 활용해 복잡한 프로그램도 빠르고 효율적으로 개발할 수 있음

강력한 확장성

데이터 분석, 인공지능(AI), 웹 개발, 업무 자동화 등 거의 모든 분야에서 활용 가능

파이썬으로 할 수 있는 일

분야 주요 라이브러리 및 프레임워크
업무 자동화 pywin32, openpyxl, xlwings, PyAutoGui
웹 크롤링 beautifulsoup, selenium
응용 프로그램 (GUI) Tkinter, PyQt5
인공지능 & 머신러닝 tensorflow, pytorch, keras, OpenCV
데이터 분석 & 시각화 pandas, numpy, matplotlib
웹(서버) 프로그래밍 Django, Flask, FastAPI
사물 인터넷(IoT) Raspberry Pi

파이썬의 한계 (할 수 없는 일)

  • 시스템 밀접 프로그래밍:

    OS 커널, 디바이스 드라이버 등 극도의 속도와 메모리 직접 제어가 필요한 영역 (C/C++의 영역)

  • 모바일 네이티브 앱:

    iOS, Android의 네이티브(Native) 앱 개발 (Kotlin, Swift의 영역)

간단한 것이라도 당장 탈 수 있는 것!
