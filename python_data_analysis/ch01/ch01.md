## 3. 파이썬 컬렉션 (자료구조)

> `자료구조.md` 참고

### 3.1 리스트 (List) - 순서O, 중복O, 수정O

```python
# 리스트 생성
icecreams = ["바닐라", "초코", "딸기"]

# 요소 추가
icecreams.append("민트초코")
print(icecreams)  # ['바닐라', '초코', '딸기', '민트초코']

# 요소 삽입
icecreams.insert(1, "녹차")
print(icecreams)  # ['바닐라', '녹차', '초코', '딸기', '민트초코']

# 요소 삭제
icecreams.remove("녹차")
print(icecreams)  # ['바닐라', '초코', '딸기', '민트초코']

# 인덱스로 접근
print(icecreams[0])  # 바닐라
print(icecreams[-1])  # 민트초코 (마지막 요소)

# 슬라이싱
print(icecreams[1:3])  # ['초코', '딸기']
```

### 3.2 튜플 (Tuple) - 순서O, 중복O, 수정X

```python
# 튜플 생성 (한번 만들면 변경 불가)
icecreams = ("바닐라", "초코", "딸기")

# 요소 접근
print(icecreams[0])  # 바닐라
print(icecreams[1])  # 초코

# 튜플 언패킹
flavor1, flavor2, flavor3 = icecreams
print(f"첫번째: {flavor1}, 두번째: {flavor2}, 세번째: {flavor3}")

# 튜플은 수정 불가 (에러 발생)
# icecreams[0] = "녹차"  # TypeError 발생
```

### 3.3 딕셔너리 (Dictionary) - 키:값 쌍, 순서O(3.7+), 중복X(키)

```python
# 딕셔너리 생성
icecream_prices = {
    "바닐라": 3000,
    "초코": 3500,
    "딸기": 3500
}

# 값 접근
print(icecream_prices["바닐라"])  # 3000
print(icecream_prices.get("초코"))  # 3500

# 값 추가/수정
icecream_prices["민트초코"] = 4000
icecream_prices["딸기"] = 4000
print(icecream_prices)

# 키 존재 확인
if "바닐라" in icecream_prices:
    print("바닐라 아이스크림이 있습니다.")

# 모든 키, 값 출력
for flavor, price in icecream_prices.items():
    print(f"{flavor}: {price}원")
```

### 3.4 세트 (Set) - 순서X, 중복X, 수정O

```python
# 세트 생성 (중복 자동 제거)
icecreams = {"바닐라", "초코", "딸기", "딸기"}
print(icecreams)  # {'바닐라', '초코', '딸기'}

# 요소 추가
icecreams.add("민트초코")
print(icecreams)

# 요소 삭제
icecreams.remove("민트초코")
print(icecreams)

# 집합 연산
store_a = {"바닐라", "초코", "딸기"}
store_b = {"딸기", "민트초코", "녹차"}

print(store_a | store_b)  # 합집합: {'바닐라', '초코', '딸기', '민트초코', '녹차'}
print(store_a & store_b)  # 교집합: {'딸기'}
print(store_a - store_b)  # 차집합: {'바닐라', '초코'}
```

### 3.5 컬렉션 비교 정리

| 자료구조 | 순서 | 중복  | 수정 | 사용 예시              |
| -------- | ---- | ----- | ---- | ---------------------- |
| 리스트   | O    | O     | O    | 학생 명단, 점수 목록   |
| 튜플     | O    | O     | X    | 좌표, 고정된 설정값    |
| 딕셔너리 | O    | X(키) | O    | 학생 정보, 설정 데이터 |
| 세트     | X    | X     | O    | 중복 제거, 집합 연산   |

## 7. 클래스

속성(상태, 데이터)과 기능을 함께 가지고 있는 객체 생성

```python
class Calculator:
    def __init__(self):
        self.value = 0
    def add(self, n):
        self.value += n
        return self.value

calc1 = Calculator()
calc2 = Calculator()

calc1.add(10)
calc2.add(100)
```

### 8.2 NumPy를 이용한 배열 연산

```python
import numpy as np

# 배열 생성
arr = np.array([1, 2, 3, 4, 5])
print(arr)  # [1 2 3 4 5]

# 배열 연산
print(arr * 2)  # [ 2  4  6  8 10]
print(arr + 10)  # [11 12 13 14 15]

# 통계 함수
print(f"평균: {arr.mean()}")  # 3.0
print(f"합계: {arr.sum()}")  # 15
print(f"최댓값: {arr.max()}")  # 5
print(f"최솟값: {arr.min()}")  # 1

# 2차원 배열 (행렬)
matrix = np.array([[1, 2, 3], [4, 5, 6]])
print(matrix)
print(f"배열 크기: {matrix.shape}")  # (2, 3)
```

### 8.3 Pandas를 이용한 데이터 분석

```python
import pandas as pd

# 데이터프레임 생성
data = {
    '이름': ['김일남', '김이남', '김삼남', '김사남'],
    '나이': [99, 98, 97, 96],
    '현금': [8500, 9200, 7800, 8800]
}
df = pd.DataFrame(data)

# 데이터 확인
print(df)
print("\n--- 기본 통계 ---")
print(df.describe())

# 데이터 필터링
print("\n--- 현금 9000원 이상 ---")
print(df[df['현금'] >= 9000])

# 데이터 정렬
print("\n--- 현금 많은 순 ---")
print(df.sort_values('현금', ascending=False))

# 평균 계산
print(f"\n평균 현금: {df['현금'].mean()}원")
print(f"평균 나이: {df['나이'].mean()}세")
```
